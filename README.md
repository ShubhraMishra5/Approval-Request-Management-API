# Approval Request Management API

A FastAPI-based backend service for managing approval workflows.  
It allows employees to create requests and enables approvers to approve, reject, or cancel them.

Designed as a reusable backend that can later integrate with frontend apps, Microsoft Teams bots, Copilot agents, or automation workflows.

---

## Features

- Create approval requests
- Approve / Reject / Cancel workflow
- Status transition validation
- Request history tracking
- Dashboard summary APIs
- Filtering and search support
- Basic API key authentication

---

## Tech Stack

- Python
- FastAPI
- SQLAlchemy / SQLModel
- SQLite
- Pytest
- Uvicorn
